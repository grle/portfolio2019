01. OVERVIEW

This website was for a nonprofit organization called Dunamis Harvest Global Ministry, which is owned by a small Korean church in Honolulu, Hawaii. Most of the people attending this church are elderly, which meant that the website should be simple in appearance and easy to navigate since most people using this site struggle with using technology.

02. LOGO

I started by updating the existing logo. I made a vector design, and cleaned up the appearance by eliminating the name of the organization encircling the logo.

03. FIRST ITERATION

I made the first iteration of the product using blue colors. It was originally supposed to be a prototype, but I took an image from UnSplash and based the color scheme off of that.

04. SECOND ITERATION

I then created a second iteration using brighter colors, and decorating it with flowers. I wanted to use a background with wheat in it to complement the harvest part of the organizationâ€™s name, and found one with poppies in it. I chose that for a bit of color and got a bit carried away with the poppy theme.

05. THIRD ITERATION

In the third iteration, I used a more modern appearance that would make this website look serious and more professional than the previous two iteration, which used a more playful layout.

06. WEBSITE

Ultimately, I switched the banner image to a picture with more darker colors to give it more gravitas. I also gave it a more professional look than the previous iterations.

07. FUTURE ADDITIONS

Future iterations of this website will include more pages in the Ministry page, which will list the locations that this nonprofit works on. At the moment, we do not have all the information needed to construct these pages, as they unfortunantly went missing. However, once the organization manages to collect the images, videos, and information they need in the website, I will be putting them in the following pages.
