01. DUNAMIS HARVEST GLOBAL MINISTRY

This website was for a Korean church in Hawaii. Most of the people attending this church are elderly, which meant that the website should be simple in appearance and easy to navigate.

I started by updating the existing logo. I made a vector design, and cleaned up the appearance by eliminating the name of the organization circling the logo.

1. FIRST ITERATION

I made the first iteration of the product using blue colors. It was originally supposed to be a prototype, but I took an image from UnSplash and based the color scheme off of that.

2. SECOND ITERATION

I then created a second iteration using brighter colors, and decorating it with flowers. I wanted to use a background with wheat in it to complement the harvest part of the organizationâ€™s name, and found one with poppies in it. I chose that for a bit of color and got a bit carried away with the poppy theme.

FINAL PRODUCT

The client went with the second design, but with the request that the flowers be removed. I included the prototype of the website, which can be viewed in the following.
